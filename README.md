# Handcrafted Haven

## Project Overview
Handcrafted Haven is an innovative web application designed to provide a platform for artisans and crafters to showcase and sell unique handcrafted items. Acting as a virtual marketplace, it connects talented creators with consumers who value sustainability and quality craftsmanship. The application emphasizes community engagement, supports local artisans, and promotes sustainable consumption.

## Technology Stack
This project utilizes a full web development stack to ensure performance, scalability, and maintainability:
*   **Front-End:** Next.js (React), HTML, CSS, JavaScript
*   **Back-End:** Node.js
*   **Database:** MongoDB
*   **Deployment:** Vercel
*   **Version Control:** Git & GitHub

## Design & Theme Strategy
The design focuses on a "consistent look and feel" that reflects the organic and artistic nature of the products sold.

### Color Palette
*   **Primary:** Earthy Green (Representing sustainability and growth)
*   **Secondary:** Terracotta/Clay (Representing raw materials and craftsmanship)
*   **Background:** Off-White (Providing a clean, neutral canvas for product images)

### Typography
*   **Headings:** *Kepler Std, or another neat serif font* - Chosen for its class and elegance.
*   **Body:** *Helvetica or another sans-serif font* - Chosen for high readability across devices.

### Accessibility & Standards
*   **Compliance:** This application is designed to comply with **WCAG 2.1, Level AA** standards.
*   **Responsiveness:** Fully responsive design ensuring seamless usage on smartphones, tablets, and desktops.

## Functional Requirements
The application includes the following core features based on the project specifications:

1.  **Seller Profiles:** Authenticated artisans can create profiles to share their stories and manage their portfolios.
2.  **Product Listings:** Sellers can upload items with descriptions, images, and pricing.
3.  **Browsing & Filtering:** Users can browse the catalog and filter items by category or price.
4.  **Reviews & Ratings:** A system for users to leave star ratings and written reviews for products.
5.  **Authentication:** Secure login for both buyers and sellers.

## Getting Started
To run this project locally:

1.  **Clone the repository:**
    ```bash
    git clone git@github.com:KamcyAhaka/handcrafted-haven.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```
4.  **Open the application:**
    Navigate to `http://localhost:3000` in your browser.
